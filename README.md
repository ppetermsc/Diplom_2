# Diplom_2 - API Тестирование Stellar Burgers

## Дипломный проект: Автоматизированное API-тестирование

## Описание проекта

Данный проект содержит автоматические API-тесты для веб-приложения **Stellar Burgers** 

**Основной сайт:** [Stellar Burgers](https://stellarburgers.education-services.ru/)
---

## Задание: Автотесты для API

Протестируй эндпоинты API для Stellar Burgers.  
Создай отдельный репозиторий для тестов API.  
Подключи **JUnit 4**, **REST Assured** и **Allure**.  
Сделай отчёт в Allure.

### ✅ Реализованная функциональность:

#### **Создание пользователя**
- ✅ Создать уникального пользователя
- ✅ Создать пользователя, который уже зарегистрирован
- ✅ Создать пользователя и не заполнить одно из обязательных полей

#### **Логин пользователя**
- ✅ Вход под существующим пользователем
- ✅ Вход с неверным логином и паролем

#### **Создание заказа**
- ✅ С авторизацией
- ✅ Без авторизации
- ✅ С ингредиентами
- ✅ Без ингредиентов
- ✅ С неверным хешем ингредиентов

## Используемые технологии

| Технология | Версия | Назначение |
|------------|---------|------------|
